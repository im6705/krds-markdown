# 엘리베이션 (Elevation)

엘리베이션은 UI에서 요소들이 서로 다른 시각적 레벨에 위치하도록 하여 깊이감과 계층 구조를 부여하는 개념이다. 색상, 그림자, 경계선, 딤드 처리를 통해 콘텐츠의 깊이감과 시각적 구분을 제공한다.

---

## 1. 엘리베이션 6단계 (표준형 스타일)

| Level | 이름 | 설명 | 적용 컴포넌트 |
|---|---|---|---|
| **-1** | 배경 아래 단계 | 배경보다 낮은 레벨. 리스트 컨테이너, 묶음형 요소 | 리스트 컨테이너 |
| **0** | Background | UI의 기본 배경 | 일반 바탕 |
| **+1** | 기본 상호작용, 비활성화 요소 | 기본 배경보다 약간 높게 표현 | official government banner, main menu, side menu, in-page navigation, tab, card |
| **+2** | 주요 상호작용 요소 | 클릭/터치 상호작용 요소, 사용자 주목도 향상 | calendar_open, main menu_open, side menu_open, select_open |
| **+3** | 주목도 높은 요소 | 상호작용 중 주목받아야 할 요소 | tooltip, open panel, bottom sheet, modal |
| **+4** | 가장 중요한 알림 | 긴급 메시지, 최상위 레벨 | critical alerts |

---

## 2. 표현 방법

| 방법 | 설명 |
|---|---|
| **그림자 (Shadow)** | 그림자 값으로 떠 있는 효과를 표현 |
| **색상 (Surface)** | 표면 색상으로 엘리베이션 표현 |
| **딤드 (Dimmed)** | 어두운 배경을 덮어 집중 효과 |
| **경계선 (Border)** | 표면을 라인으로 구분 |

---

## 3. 그림자 표현 (4단계)

그림자는 X, Y 오프셋, 블러 반경, 투명도를 조정하여 시각적 깊이감을 표현한다. 기본 모드에서 효과적이며, 선명한 모드에서는 표면 밝기와 함께 사용한다.

| 단계 | 용도 | 예시 |
|---|---|---|
| **shadow1** | 기본 상호작용 요소에 얕은 깊이감 | 기본 버튼, 카드 |
| **shadow2** | 주요 상호작용 요소에 뚜렷한 그림자 | 드롭다운, 메인 메뉴, 셀렉트 |
| **shadow3** | 강한 깊이감 | 툴팁, 패널 |
| **shadow4** | 가장 강한 그림자, 사용자 주목 필요 | 모달, 긴급 알림 |

### 엘리베이션 적용 예시

| 컴포넌트 | 그림자 스타일 |
|---|---|
| 셀렉트 | shadow2 |
| 툴팁 | shadow3 |
| 모달 | shadow4 |
| 긴급 공지 | shadow2 + 고도 단계 -1의 surface |

---

## 4. 색상으로 엘리베이션 표현

### 레이어드 (Layered)

여러 표면이 겹쳐질 때 단계별 색상 차이를 둔다.

| 모드 | 방식 |
|---|---|
| **기본 모드** | 흰색과 색상 표면을 번갈아 사용 |
| **선명한 화면 모드** | 단계별로 점차 밝아지도록 조정 |

### 선명한 화면 모드 표면 밝기 3단계

| 단계 | 토큰 | 용도 |
|---|---|---|
| **1단계** | surface-white | 배경과 동일한 기본 레벨, 시각적 통일감 |
| **2단계** | surface-white-subtle | 더 밝은 표면으로 1단계와 대비, 깊이감 표현 |
| **3단계** | surface-white-subtler | 가장 밝은 표면으로 최상위 레벨 강조 |

---

## 5. 딤드 효과

딤드는 특정 요소에 집중시키기 위해 배경을 어둡게 처리하여 시각적 대조를 제공한다.

### 사용 규칙

- **과도한 사용을 피한다**: 주의가 필요한 요소에만 제한적으로 사용
- **중첩된 딤드 효과 지양**: 여러 겹의 딤드는 시각적 혼란 초래, 단일 효과로 제한
- **조작 가능한 영역 고려**: 딤드 상태에서도 닫기 버튼 등 상호작용 영역은 명확하게 표시

---

## 6. 경계선을 이용한 구분

선명한 화면 모드에서는 그림자가 잘 드러나지 않으므로 경계선으로 콘텐츠 간 시각적 구분을 보조한다.

- **얕은 엘리베이션 단계**: 그림자가 미세할 때 경계선 추가로 요소를 배경과 구분
- **다중 콘텐츠**: 카드/패널이 중첩된 경우 경계선으로 요소 간 구분 명확화
