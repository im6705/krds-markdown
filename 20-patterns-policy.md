# 정책 정보 확인 서비스 패턴

정책 정보 확인은 디지털 서비스에 게재된 정부/기관의 행동 방침, 계획, 법률에 관한 정보를 사용자가 확인하는 과업이다.

---

## 1. 정보 유형

| 유형 | 설명 | 예시 |
|---|---|---|
| **정책 정보** | 정부가 공공문제를 해결하거나 정치적·행정적 목표를 실현하기 위해 마련한 법률·정책·사업·사업계획·정부방침·정책지침·결의 사항 등에 대한 정보 | 주요 정책 소개 |
| **관련 자료** | 주요 정책을 이해하고 참여하는데 필요한 각종 자료 | 보고서, 매뉴얼, 법령 자료, 정책 홍보자료, 연구 자료, 국정 과제 등 |

---

## 2. 이용 상황별 플로(Flow)

| 단계 | 과정 | 설명 |
|---|---|---|
| 1 | **정책 상세 정보 확인** | 상세 콘텐츠를 확인하여 찾고자 한 내용에 부합하는지, 어떤 내용이 포함되어 있는지 등을 이해하고 비교 및 검토를 수행하는 과정 |
| 2 | **정책 자료 참고** | 정책과 관련된 각종 자료를 통해 궁금증을 해결하거나 사업·서비스의 참여에 도움이 되는 내용을 확인하는 과정 |

---

## 3. 단계별 상세

### 3.1 정책 탐색

#### 구조

| # | 요소 | 설명 |
|---|---|---|
| 1 | **필터링·정렬 컨트롤** | 정책 목록을 필터링·정렬하는 컨트롤 |
| 2 | **페이지네이션** | 정책 목록 탐색용 컨트롤 |
| 3 | **항목** | 정보 식별을 위한 콘텐츠 집합 |
| 3-a | 제목 | 정책명 (상세 화면 이동 링크) |
| 3-b | 미리보기/요약 | 정책 기본 정보 요약 텍스트 |
| 3-c | 꺾쇠/화살표 | 링크 동작을 안내하는 시각적 단서 |
| 3-d | 배지 | 정책의 주요 분류 체계를 나타내는 메타 데이터 |
| 3-e | 메타 데이터 | 배지 외 데이터 속성 (분야, 기간 등) |
| 3-f | 저장 컨트롤 | 관심 정책 정보 모아보기용 개인화 저장 |

#### 사용성 가이드라인

| # | 가이드라인 | 적용 수준 |
|---|---|---|
| 01 | 모든 정책을 하나의 목록에서 탐색할 수 있는 수단을 제공한다 | 권장 |
| 02 | 필터링 또는 검색 기능을 제공한다 | 권장 |
| 03 | 정책의 대상/분야 등의 분류 체계를 표시한다 (배지, 메타 데이터 활용) | 권장 |
| 04 | 새로운 정책 및 자료를 명확하게 구분한다 ('신규', '중요' 배지) | 권장 |

#### 접근성 가이드라인

- 공유/저장 컨트롤과 액션 버튼에 명확한 접근 가능한 이름을 제공한다 — `title` 또는 `aria-describedby` 활용 (KWCAG 2.2 적절한 링크 텍스트, WCAG 2.1 Label in Name (A), Name, Role, Value (A))

---

### 3.2 정보 확인

#### 사용성 가이드라인

| # | 가이드라인 | 적용 수준 |
|---|---|---|
| 01 | 화면 제목을 정책명으로 제공한다 | 권장 |
| 02 | 상세 정보 콘텐츠는 간결하며 이해하기 쉽게 작성한다 (전문 용어 사용 시 도움 컴포넌트 활용) | 권장 |
| 03 | 상세 정보는 정확하고 최신화된 상태로 제공한다 | 필수 |
| 04 | 내용이 없는 콘텐츠 섹션과 섹션 제목은 표시되지 않도록 한다 | 권장 |
| 05 | 인포그래픽, 사진, 동영상 등의 콘텐츠를 적절하게 활용한다 | 우수 |
| 06 | 사용자에게 도움을 줄 수 있는 다양한 부가 정보를 제공한다 | 권장 |
| 07 | 부가 정보는 사용자가 필요에 따라 상세 내용을 확인할 수 있게 제공하는 방안을 고려한다 | 권장 |
| 08 | 모든 링크는 실행하였을 때, 링크 레이블에 명시된 적절한 화면으로 이동해야 한다 | 권장 |

**제공 가능한 부가 정보 예시:**

- 정책에 대한 문의처 (담당자 이름, 전화번호, 이메일)
- 관련 정책 링크 목록
- 관련 자료 다운로드 링크, 자료 게시판
- 관련 웹사이트 링크, 정책 관련 신청 링크

| 3 | **항목** | 정보 식별을 위한 콘텐츠 집합 |
| 3-a | 제목 | 자료명 (상세 화면 이동 링크) |
| 3-b | 미리보기/요약 | 자료 기본 정보 요약 텍스트 |
| 3-c | 꺾쇠/화살표 | 링크 동작을 안내하는 시각적 단서 |

#### 사용성 가이드라인

| # | 가이드라인 | 적용 수준 |
|---|---|---|
| 01 | 필터링 또는 검색 기능을 제공한다 (기간, 자료 유형 등) | 권장 |

| 03 | 정책 관련 간행물의 발행 주기에 대한 정보를 제공한다 (변경/중단 상황 포함) | 권장 |
