## 핵심 수치 참조표

### 브레이크포인트 (표준형 4단계)

| name | Viewport | 칼럼 수 | 가터 | 스크린 마진 |
|---|---|---|---|---|
| small | 360px~ | 4 | 16px | 16px |
| medium | 768px~ | 8 | 16px | 24px |
| large | 1024px~ | 12 | 24px | 24px |
| xlarge | 1280px~ | 12 | 24px | 24px |

### 매직넘버 (명도 대비)

| 매직넘버 | 대비 비율 | WCAG 기준 |
|---|---|---|
| 40 | 3:1 | AA (대형 텍스트/UI) |
| 50 | 4.5:1 | AA (일반 텍스트) |
| 70 | 7:1 | AAA |
| 90 | 15:1 | 최고 수준 |

### 래디어스 (표준형 5단계)

| Level | Radius | Container Size 범위 |
|---|---|---|
| Xsmall | 2px | 8~16px |
| Small | 4px | 20~32px |
| Medium | 6px | 40~64px |
| Large | 10px | 72~80px |
| Xlarge | 12px (max) | 96~120px |

### 타이포그래피 기본

| 항목 | 값 |
|---|---|
| 기본 서체 | Pretendard GOV |
| 기본 본문 크기 | 17px |
| 최소 본문 크기 | 16px |
| 글자 두께 | regular(400), bold(700) |
| 줄 간격 | 150% 이상 |
| rem 기본값 | 10px (62.5%) |

### 색상 비율

| 비율 | 역할 |
|---|---|
| 60% | 배경 및 중립적인 색상 |
| 30% | 보조 색상 |
| 10% | 주요 기능/상호작용 색상 |
| 최대 5% | 강조 색상 (accent) |

### 적용 수준 (서비스 패턴)

| 수준 | 설명 |
|---|---|
| 필수 (Do) | 미준수 시 작업 실패 |
| 권장 (Better) | 미준수 시 작업 난도 증가 |
| 우수 (Best) | 준수 시 만족도 기하급수적 증가 |

## 관련 법령

- 전자정부법
- 행정기관 및 공공기관 정보시스템 구축/운영지침

- WCAG 2.1 (적합도 수준 AA 목표)
