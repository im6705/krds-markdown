# 신청 서비스 패턴

신청은 사용자가 서비스 제공 기관에 요구 사항을 알리기 위해 서식을 작성하는 과업으로, 관련 법률에 명시된 종류의 민원 외에 사용자가 특정한 내용을 요청하는 모든 행동을 포함한다.

---

## 1. 신청 유형

| 유형 | 설명 | 예시 |
|---|---|---|
| **제출** | 정해진 서식에 따라 신청서를 작성하여 제출 | 복지 급여 신청, 전입신고, 귀국신고 |
| **조회** | 자격, 문서 효력, 처분 결과 등을 확인 | 문서 진위 확인, 세액 조회 |
| **발급** | 증명 서류, 자격 번호 등을 발급 | 주민등록 등·초본 발급, 개인통관번호 발급 |
| **예약** | 일정 기간 동안의 재화/서비스에 대한 이용 신청 | 도서 예약, 좌석 예약, 관람 예약/예매 |

---

## 2. 사용자 여정 (4단계 플로)

### 전체 플로

1. **신청 대상 탐색**
   - 서비스 목록 탐색: 디지털 서비스에서 제공 중인 신청 서비스 목록에서 발견
   - 대상 선택하기: 한 번에 여러 건 선택 및 일시 저장
   - 서비스 정보 확인: 상세 정보 확인
   - 유의 사항/자격 확인: 주의해야 하는 사항 안내, 자격 여부 확인

2. **신청서 작성**
   - 단계의 진행과 탐색: 여러 단계로 구성된 신청서 작성
   - 신청 정보 입력: 서식에서 요구하는 항목에 데이터 입력
   - 신청 정보 자동 입력: 인증에서 확인된 정보 또는 기존 데이터 활용
   - 서류 제출: 증빙 서류(증명 사진, 주민등록등·초본 등) 첨부
   - 작성 도움말 확인 / 매뉴얼 확인
   - 임시 저장: 제출 이전에 일정 기간 보관 (명시적 저장 또는 자동 저장)

3. **확인/확정**
   - 신청 내용 최종 확인: 작성한 내용이 정확한지 검토
   - 인증: 신청 완료 이전에 최종적으로 사용자 신원 확인
   - 신청서 제출 확정: 최종 단계에서 신청 행위를 확정
   - 신청서 작성 취소: 최종 단계에서 신청 행위를 철회
   - 여러 건 신청하기: 동일 서비스를 다양한 조건으로 반복

4. **신청 결과 확인**
   - 신청서 작성 완료: 신청서 제출 후 완료 확인
   - 결과 화면 찾기: 제출 이력 확인 서비스 탐색
   - 결과 목록 탐색: 신청 이력을 일자별/진행 상태별로 확인
   - 진행 상태 확인: 기관/시스템의 처리 상황 확인

---

## 3. 신청 대상 탐색 - 구조

| # | 요소 | 설명 |
|---|---|---|
| 1 | **필터링/정렬 컨트롤** | 신청 서비스 목록을 필터링/정렬 |
| 2 | **페이지네이션** | 목록 탐색용 컨트롤 |
| 3 | **항목** | 정보 식별을 위한 콘텐츠 집합 |
| 3-a | 제목 | 서비스명 (상세 화면 이동 링크) |
| 3-b | 꺾쇠/화살표 | 링크 동작을 안내하는 시각적 단서 |
| 3-c | 미리보기/요약 | 서비스 기본 정보 요약 |
| 3-d | 배지 | 이용 가능 상태 (예정됨/접수 중/마감됨) |
| 3-e | 메타 데이터 | 데이터 속성 (분야, 연령, 소득 등) |
| 3-f | 저장 컨트롤 | 관심 정보 모아보기 |
| 3-g | 액션 버튼 | 신청하기 버튼 등 |

---

## 4. 신청 대상 탐색 - 사용성 가이드라인

| # | 가이드라인 | 적용 수준 |
|---|---|---|
| 01 | 신청 서비스의 특성에 따른 적절한 서비스 목록 정렬 방식을 사용한다 (상시: 가나다순 / 한시적: 신청 가능 우선) | 필수 |
| 02 | 목록을 적절한 방식으로 군집화하여 메타 데이터를 부여한다 | 필수 |
| 03 | 신청 상태 정보를 명확하게 인지할 수 있도록 배지를 반드시 제공한다 | 필수 |
| 04 | 미리보기/요약에 신청 가능 기한 정보를 제공한다 | 권장 |
| 05 | 제목에 공식적인 서비스 명칭을 사용한다 | 필수 |
| 06 | 제목에 말줄임표를 사용하지 않는다 | 필수 |
| 07 | 미리보기/요약 텍스트는 간결하게 작성한다 | 권장 |
| 08 | 상세 화면을 거치지 않고 신청 과정에 바로 접근할 수 있는 액션 버튼을 제공한다 | 우수 |
| 09 | 외부 서비스로 이동하거나 새 창을 실행하는 액션 버튼에 명확한 시각적 단서를 제공한다 | 필수 |

---

## 5. 관련 컴포넌트

- **컴포넌트**: 구조화 목록, 링크, 페이지네이션
- **기본 패턴**: 필터링/정렬, 목록 탐색
